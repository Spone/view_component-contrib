## Installation

Adding to a gem:

```ruby
# my-cool-gem.gemspec
Gem::Specification.new do |spec|
  # ...
  spec.add_dependency "view_component-contrib"
  # ...
end
```

Or adding to your project:

```ruby
# Gemfile
gem "view_component-contrib"
```

### Supported Ruby versions

- Ruby (MRI) >= 2.5.0
- JRuby >= 9.2.9
